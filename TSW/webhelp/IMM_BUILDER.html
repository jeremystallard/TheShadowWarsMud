IMM_BUILDER
The following is a list of commands for builders to use as a reference.  

 

alist - lists all the areas, vnums and builders

 

vlist obj   - lists all objs between vnum values


vlist mob   - lists all mobs between vnum values


vlist room   - lists all rooms between vnum values

 

Also see:

[ help olc ] for directions on how to build.

[ help hitdice ] for a standard range of mob hitpoints.

[ help damdice ] for a standard range of mob damage.

[ help mana ] for a standard range of mob endurance.

[ help hitroll ] for a standard range of mob hitrolls.

[ help armor_class ] for a standard range of mob armor classes.

[ help weapons_g1 ] for a standard range of weapon damage and info.

[ help armor_g1 ] for information about building armor.

[ help costs ] for information about pricing various items.

[ help reset ] for information about reseting your rooms.
