WRITE READ SEALLETTER UNSEAL LETTER LETTERS
Requires: A piece of paper held in your hand...

Syntax: Write  - writes a message onto the piece of paper

Syntax: write write - puts you in text editor mode and then copies the contents

of the message to the note when you exit edit mode

 

Syntax: Read  - Reads the contents of the note

Syntax: Seal  - Seal the note so others can't read it

Syntax: Unseal  - Break the seal on the note, so that others can read it
