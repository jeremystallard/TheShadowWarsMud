PC_SELECTION_MENU
This menu give an overview of your character creation and allow you to

more detailed set up you background information so that it might fit with

the Role Play you have planned for this character.  

 

Available options in this menu:

o Spheres      - Channelers can tune their spheres in this menu.

o Stats        - All character can tune their stats in this menu.

o Merits       - Select merits for your character in this menu.

o Flaws        - Select flaws for your character in this menu.

o Talents      - Select talents for your character in this menu.

o Help         - This help entry.

o Help  - Help on any command, skill.

 

Type one of the available options to enter the sub-menus.

 

When you are satisfied with your selections type 'done' to continue.

 

[Hit Return to continue]
